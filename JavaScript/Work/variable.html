Variable Declaration & Initialization


JavaScript: uses three main Keywords to declare variables:

Keyword Description
let Modern way (block-scoped).Preferred for values that will change
reassignable
const Constant (block-scoped). preferred for values that won't change (but object properties can change)
var Old way (function-scoped).Avoid using in modern JS. 

 ⭢   Example:
let age = 25;                      //Mutable 

const PI = 3-14;                   //Immutable (reassignment throws error)  

varname  = "Alice";                //Avoid (leaks scope)
